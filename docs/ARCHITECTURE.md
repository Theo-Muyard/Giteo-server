# Architecture

## Commandes
- `help`
Affiche de l'aide sur toutes les commandes, de l'aide specifique est aussi donnee.
- `adduser`
Ajoute un utilisateur aux utilisateurs ayant acces au systeme, cet utilisateur pourra se connecter a giteo sur ce serveur specifique.
- `removeuser`
Supprime un utilisateur aux utilisateurs ayant acces au systeme, cet utilisateur n'aura donc plus acces a giteo sur ce serveur specifique.
- `setup`
Permet de creer les fichiers de configuration de base ainsi que de creer le premier utilisateur.
